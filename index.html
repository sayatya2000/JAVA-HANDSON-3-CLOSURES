4.Write a code to calculate area of a rectangle using inner function. In this case outer function should accept parameter length and inner function should accept parameter breadth.

Ans.

      function calculateArea(length) {
        function getBreadth() {
          return 8; // hardcoded breadth value for example purposes
        }
        var breadth = getBreadth();
        var area = length * breadth;
        return area;
      }
      console.log(calculateArea(10)); // output: 80

Explanation: This code defines an outer function `calculateArea` that accepts a `length` parameter, and an inner function `getBreadth` that returns a hardcoded breadth value. The outer function calls the inner function to retrieve the breadth value, and uses it to calculate the area of the rectangle by multiplying the length and breadth values. The calculated area is then returned from the function.
